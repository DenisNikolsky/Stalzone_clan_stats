# 🎮 Clan Manager for Stalcraft

Приложение для управления кланом в игре Stalcraft с поддержкой OCR-распознавания ников из скриншотов, ведения статистики и автоматического начисления предупреждений.

---

## 📋 Описание

**Clan Manager** — это десктопное приложение на Python с графическим интерфейсом, которое помогает администраторам кланов в Stalcraft управлять составом, отслеживать посещаемость событий и контролировать дисциплину. Программа использует Tesseract OCR для распознавания ников и реальных имён игроков с фотографий (скриншотов), что ускоряет заполнение базы данных.

### Основные возможности:
- ✅ Добавление участников вручную или массово через фото (OCR)
- ✅ Проведение событий (Турнир, Gold Drop, Потасовка) с отметкой присутствующих
- ✅ Автоматическое начисление предупреждений за пропуски
- ✅ Ручное управление предупреждениями (выдача, удаление, редактирование причины)
- ✅ Полная история событий и предупреждений
- ✅ Сортировка и множественное выделение в таблице
- ✅ Контекстное меню (ПКМ) для быстрых действий
- ✅ Импорт/экспорт данных в Excel (поддержка .xlsx)
- ✅ Приятная тёмная тема с настраиваемым шрифтом
- ✅ Статусная строка с количеством участников в клане
- ✅ Портативный .exe файл со встроенным Tesseract (не требует установки)

## 🚀 Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/yourusername/clan-manager-stalcraft.git
   cd clan-manager-stalcraft
   ```
2. **Запуск:** <br>
Перейдите в папку dist и запустите ClanManager

**ВАЖНО!!!**
При запуске создадится база данных Clan в том месте, где находится Clanmanager. Не паникуйте!

Профит!

**Пы.Сы.**
Для добавление участников в клан и события советую использовать белый фон и черный текст для лучшего распознавания
